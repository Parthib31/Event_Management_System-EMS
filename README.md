# EMS - Event Management System
---

## Introduction

Created a EMS web software to handle the event management system of Institute. It is a Full stack React application with MERN stack. It ensures a smooth user experience and both admin and user can logged in and have their job done.


---
## Objective

The main purpose of this software is to automate the event mangement process and less humnan intervention is required.

## Tech Stacks Used

### Frontend:

* React.js
* Bootstrap
* HTML
* CSS
* Javascript

### Backend:

* Node.JS
* Express.JS
* MongoDB
* Mongoose
* Bcrypt.js

## Procedure
- Fork the repository and clone the project in your local machine.
- In a terminal, run `npm install` to download all the dependencies.
- Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
